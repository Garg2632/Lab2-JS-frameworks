# Node.js Assignment - ES Modules

This assignment demonstrates basic usage of **custom JavaScript modules** using **Node.js with ES Module syntax**.

### Contents

- `app.js`: Entry file that imports and calls functions from `utils.js`
- `utils.js`: Contains 5 functions:
  1. `sendStudyReminder`: Prints a study reminder using user input
  2. `litresToGallons`: Converts litres to gallons
  3. `calculateAverage`: Computes average of a list of numbers
  4. `formatCountdown`: Formats seconds into `HH:MM:SS` format

### How to Run

1. Make sure Node.js is installed.
2. Run the following command in terminal:

```bash
node app.js
